Gérer ses fichiers avec GitHub

1. Créer un dépôt GitHub

Créez un nouveau dépôt dans GitHub, choisissez de le rendre privé ou public selon vos besoins.

2. Initialiser Git en local

Utilisez la commande suivante pour initialiser Git dans votre répertoire local :

git init

3. Ajouter et envoyer des fichiers

git add .
git commit -m "Premier commit"
git push origin master

4. Synchronisation sur un autre PC

git clone https://github.com/votre-nom-utilisateur/nom-du-depot.git

5. Gestion des versions

Utilisez GitHub pour gérer plusieurs versions d’un fichier.
